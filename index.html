A new way to embed music

Orquidea is a modern web library for embedding music on the web using a simple tag system

Simple

Orquidea is simple set up. Only include jQuery + plus the required .js and .css files, and add some data attributes tags to have a fully working play widget in minutes that handles the resolution iself!
